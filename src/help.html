Application Overview

The NGVS Search tool has three main components:

    Map Pane Sidebar Overlay Pane

Map Pane

The Map pane contains an interactive

Sidebar

Location Search

Location Search uses the CADC target name resolver to plot the location of one or more objects as markers on the map. Multiple markers can be plotted simultaneously by separating object names by spaces.

Comma separated RA/Dec pairs in either decimal or HMS/DMS format is also supported.

To remove search markers submit an empty search.

Example. Add three markers to map: m84 12:30:01, 12:01:04 m87
Query Catalogs

The Query Catalog section can be used to plot objects from various catalogs in the map pane. Catalog information can also be downloaded in csv format. To apply a catalog query on the map, select the catalog from the drop-down menu and click apply. Note that very large catalogs may slow down the browser. It is reccomended to refine queries in these cases. To download the catalog queries in csv format, click the download button.

Refining queries

After selecting a catalog, a list of various parameters are displayed that can be used to refine the objects that are displayed on the map (or included in the download).

Numerical ranges are entered by separating the lower limit from the upper limit separated by two periods.

Example. Restrict the query to RA values between 187 and 188: "187..188"

Single inequalities are entered by using the operator followed by the value.

Example. Restrict the query to class_spec values of 3 and above: >= 3 or > 2.

Equalities are entered by entering the value alone.

Example. Restrict query to objects with a nucleus of 1: 1

Multiple refinements for each parameter can be added by separating the refinements by commas.

Example. Multiple RA restrictions: 187..188, >= 190
FITS Images

All of the available FITS images for a region can be listed my holding shift while clicking and dragging a box around the desired region in the map pane. Alternatively all FITS images available in the current map pane view can be listed by clicking "List Images In Current Map View".

Adjust Map

The Adjust Map section contains sliders that can adjust the display of background tiles in the map pane. This can be used to enhance visibility of objects or markers.

Overlay Pane

The overlay pane is located in the top right section of the map pane. It contains groups of checkboxed that are used to toggle the visibility of various map overlays.

Base Maps

The Base Map section toggles the background images on the map. By default the map shows NGVS images where available with SDSS images covering the rest of the map. Adjustments to the background images can be made using the adjust map tab in the sidebar.

Pointings

The pointings section displays the name and outlines of the NGVS pointings. Note that any field outline overlays will render on top of the NGVS pointing outlines.

Field Outlines

The field outline sections display the outlines where FITS images are available for each filter. There are separate sections for long and short exposures. Outlines where stacked images are available are shown as bold solid lines. Outlines where single images are available are shown as dotted lines.

The field outlines are applied in the order that the checkboxes are toggled. In the case where two outlines are overlapping, the one that was most recently toggled on the map will be displayed.

Queries

The query section is used to quickly toggle visibility of any catalog queries that have been applied on the map. By default the map has no queries applied, and the section will not be visible.
